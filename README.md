# Full Stack Open – University of Helsinki

This repository contains my solutions for the **Full Stack Open** course by the University of Helsinki.

## Structure

- `part0/` – Fundamentals of Web Apps (diagrams)
- `part1/` – Introduction to React
- `part2/` – Communicating with server
- `part3/` – Programming a server with Node.js and Express
- `part4/` – Testing Express servers, user administration
- `part5/` – Testing React apps
- `part6-13/` – Advanced topics and extensions

Each part has its own directory following the course instructions.
